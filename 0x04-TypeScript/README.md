# 0x04-TypeScript

Welcome to the **0x04-TypeScript** project! This project focuses on learning and applying the fundamentals of TypeScript in a modern JavaScript development workflow. It's part of the **ALX Frontend Curriculum**, aiming to build strong foundations in statically typed JavaScript programming.

---

## 📌 Project Objectives

- Understand the core concepts of TypeScript.
- Learn how TypeScript improves code safety and readability.
- Use type annotations for variables, functions, and objects.
- Work with interfaces and classes in TypeScript.
- Integrate TypeScript with tools like Webpack and Babel.

---

## 🛠️ Technologies Used

- **TypeScript** (v4+)
- **JavaScript (ES6+)**
- **Webpack** for module bundling
- **Babel** for transpiling
- **Node.js / npm**
- **VS Code / Terminal**

---

## 🧠 Key Concepts Covered

- Basic type annotations
- Interfaces and types
- Classes and inheritance
- Enums and Generics
- Modules and Namespaces
- Type casting and assertions
- Working with DOM in TypeScript
